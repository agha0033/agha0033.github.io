5 HTML tags I have learned so far:

Abbriviation Tag <abbr>
Used for acronyms or abbreviations - when you hover over the word on the live page it shows the expanded name/full name of the acronym or abbreviation.
Line Break Tag <br>
This is a singular tag that creates a line break for content. It is recommended not to be used repeatedly in your code unless needed.
Heading Tag (ex. <h1> </h1>)
Used for creating headings for websites; <h1> would be the main heading and the heading after that are (h2, h3, h4, h5, h6) based on the numbers. The higher headings will be decreased in size and emphasis.
Image Tags <img>
Used for displaying images on a webpage. It needs informatoin form two attributes: source (src), which provides the path to the image, and alternative text (alt), which provides a description for the image.
Table Tags <table> </table >
Used for creating tables in HTML, the container. More tags are are needed in order to make rows and input all the data:<caption ></caption > - for the description/label of table; <thead ></thead > - for the header of the table; < tbody> </tbody > - for the body of the table; <tr ></tr > - for adding rows; and so on.
